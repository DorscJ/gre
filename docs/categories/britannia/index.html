Roman Palace in Fishbourne

Province Britannia Province Description Location Fishbourne Fishbourne (Pleiades) Location Description Sublocation Sublocation Description Garden Roman Palace in Fishbourne Keywords colonnades palaces piscinae Garden Description Fishbourne Roman Palace was accidentally discovered during the digging of a water main trench in 1960. The discovery led to nine seasons of excavations (1961-69) that showed the site had developed from a military base at the time of the Roman invasion in AD 43 to a sumptuous palace by the end of the first century CE.

Suburban Gardens in Camulodunum

Province Britannia Province Description Location Camulodunum Camulodunum (Pleiades) Location Description Camulodunum began as an Iron Age settlement defended by dykes, a combination of V-shaped ditch and rampart created by the upcast earth. It extended from the Roman river on the south to the Colne on the north, a distance of some four kilometers. Within this larger area, a legionary fort was built after the Claudian conquest of Britain in AD 43.
