Roman Palace at Fishbourne

Province Britannia Province Description Location Fishbourne Location Description Sublocation Sublocation Description Garden Roman Palace at Fishbourne Keywords colonnades palaces piscinae Garden Description Fishbourne Roman Palace was accidentally discovered during the digging of a water main trench in 1960. The discovery led to nine seasons of excavations (1961-69) that showed the site had developed from a military base at the time of the Roman invasion in AD 43 to a sumptuous palace by the end of the first century CE.
