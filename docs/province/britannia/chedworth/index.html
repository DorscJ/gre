Rural Villa in Chedworth

Province Britannia Province Description Location Chedworth Chedworth (Pleiades) Location Description Sublocation Sublocation Description Garden Rural Villa in Chedworth Keywords bathhouses (water recreation structures) mosaics (visual works) sherds villae rusticae Garden Description This Roman villa in Gloucestershire was partially excavated in the late 19th century, but only the built features and their mosaic floors appear to have been of interest to the Victorian diggers. Excavations in the courtyard conducted between 2000 and 2002 revealed that earlier excavators had not disturbed the courtyard.
